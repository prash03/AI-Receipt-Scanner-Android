<h1 align="center">💸 AI Receipt Scanner</h1>

<p align="center">
🤖 <b>AI-powered Expense Tracker App</b><br/>
Scan receipts → Extract data → Track expenses automatically
</p>

<p align="center">
🚀 Built with Kotlin | Jetpack Compose | ML Kit | CameraX
</p>

<hr/>

<h2>🔥 Problem</h2>

<p>
Managing expenses manually is difficult:
</p>

<ul>
<li>📄 Receipts get lost</li>
<li>⌨️ Manual entry is time-consuming</li>
<li>❌ High chances of human error</li>
</ul>

<p>
Most apps require users to <b>type everything manually</b>, making the process inefficient.
</p>

<hr/>

<h2>💡 Solution</h2>

<p>
This app uses <b>on-device AI</b> to automate expense tracking:
</p>

<ul>
<li>📷 Scan receipt using camera</li>
<li>🤖 Extract text using ML Kit</li>
<li>💰 Identify amount, date & merchant</li>
<li>📊 Save and track expenses</li>
</ul>

<p>
⚡ No backend required — everything works <b>offline</b>
</p>

<hr/>

<h2>🚀 Features</h2>

<ul>
<li>✨ AI-based receipt scanning</li>
<li>📷 CameraX integration</li>
<li>⚡ Fast text extraction (ML Kit)</li>
<li>📋 Copy & edit extracted data</li>
<li>📊 Expense tracking (planned)</li>
<li>🔒 Privacy-friendly (on-device processing)</li>
</ul>
